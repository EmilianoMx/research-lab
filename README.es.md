## Inicio rápido (para personas sin perfil técnico)
1. Instala Git y Python 3.
2. Clona el repositorio:
   ```bash
   git clone <tu-url-del-repo>
   cd research-lab
   ```
3. Ejecuta la validación incluida:
   ```bash
   make validate
   ```
4. Si prefieres inglés, abre `README.en.md`.
