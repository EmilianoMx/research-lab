# Checklist de Reproducibilidad Científica

## Información general
- [ ] Título del estudio:
- [ ] Nombres de autoría:
- [ ] Afiliaciones:
- [ ] Contacto de autor/a corresponsal:

## Disponibilidad de datos
- [ ] ¿Todos los datos necesarios para reproducir hallazgos están disponibles?
- [ ] ¿Los datos están compartidos en un formato claro y accesible?
- [ ] ¿Existen restricciones de acceso? (si aplica, justificar)

## Metodología
- [ ] ¿Los métodos están descritos con detalle suficiente para replicación?
- [ ] ¿Se especifican software y versiones?
- [ ] ¿Se documentan parámetros y configuraciones del análisis?
- [ ] ¿Existe un diseño experimental claramente descrito?

## Código y análisis
- [ ] ¿El código utilizado está disponible?
- [ ] ¿El código está comentado de forma clara?
- [ ] ¿Dependencias y entorno están especificados?
- [ ] ¿Existe flujo/pipeline documentado?

## Reporte de resultados
- [ ] ¿Los resultados se presentan de forma clara y precisa?
- [ ] ¿Figuras y tablas están etiquetadas y descritas adecuadamente?
- [ ] ¿Los análisis estadísticos están informados con métodos y resultados?

## Revisión y retroalimentación
- [ ] ¿El trabajo ha sido revisado por pares?
- [ ] ¿Se documentaron respuestas a comentarios de revisión?

## Consideraciones éticas
- [ ] ¿Se obtuvieron aprobaciones éticas cuando corresponde?
- [ ] ¿Se declararon conflictos de interés?

## Proceso de publicación
- [ ] ¿Existe plan claro de revista/conferencia objetivo?
- [ ] ¿Se evaluaron opciones de acceso abierto?

## Notas adicionales
- [ ] Cualquier información adicional relevante para reproducibilidad.
