# Estándares Metodológicos para Publicaciones Q1

## Introducción
Este documento resume estándares metodológicos orientados a publicaciones Q1. Su adopción fortalece calidad, credibilidad y robustez de resultados.

## 1. Diseño de investigación
- **Claridad de preguntas e hipótesis:** Formular preguntas de investigación precisas y verificables.
- **Diseño adecuado:** Seleccionar un diseño (experimental, observacional, mixto) coherente con objetivos y preguntas.

## 2. Tamaño muestral
- **Cálculo de tamaño de muestra:** Estimar poder estadístico para detectar efectos esperados.
- **Representatividad:** Asegurar representatividad de la población objetivo para mejorar generalización.

## 3. Recolección de datos
- **Instrumentos estandarizados:** Usar instrumentos validados para reducir sesgo.
- **Capacitación del equipo:** Formar y monitorear a quienes recolectan datos.

## 4. Análisis de datos
- **Métodos estadísticos apropiados:** Elegir métodos según tipo de variable, distribución y supuestos.
- **Transparencia de software:** Reportar software y versión utilizada para análisis.

## 5. Reporte
- **Reporte transparente:** Seguir guías como CONSORT, PRISMA o STROBE según el tipo de estudio.
- **Tamaños de efecto:** Informar tamaños de efecto junto con significancia estadística.

## 6. Consideraciones éticas
- **Aprobación y consentimiento:** Obtener aprobaciones éticas y consentimiento informado cuando corresponda.
- **Transparencia financiera:** Declarar financiamiento y potenciales conflictos de interés.

## Conclusión
Estos estándares representan un mínimo esperado para investigación reproducible y publicable en revistas de alto impacto.

- Moher, D. et al. (2010). *PRISMA Statement*.
