## Quick start (for non-programmers)
1. Install Git and Python 3.
2. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd research-lab
   ```
3. Run the built-in validation:
   ```bash
   make validate
   ```
4. Open `README.es.md` if you prefer Spanish documentation.
